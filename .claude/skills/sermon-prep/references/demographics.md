## How to Use This File

When preparing a sermon, reference this file during the brainstorm phase to answer:

- Who is the person in the third row who needs this sermon most?
- What is the audience need this passage addresses?
- What illustration or application will land given where these people are?

Update this file whenever your congregation's makeup or circumstances shift significantly.
