# First Presbyterian Church of Griffin — Brand Guidelines

**Edition:** Volume 01 · 2026 · For Internal Use + Partners

---

## Mission & Identity

**Mission:** To invite hungry people to be filled with joy in Christ Jesus.

**Tagline:** Know joy. Know love. Know Jesus.
- Always on its own line, set in the display face
- Final clause ("Know Jesus.") visually distinct — italicized or in Brand Blue
- Never rearrange the order

**Identity Principles:**
- **Reformed, never cold.** Serious about truth. Warm about people. Design should feel the same.
- **A family, not a venue.** Prefer plural, relational language. Avoid event-marketing gloss.
- **Hope is the headline.** Everything — poster to bulletin — should point to Christ.

### Usage Ratios (for full-page layouts)

- **Light `#F4F7F8` — 60%** (dominant background)
- Light Gray `#D9E1E4` — 15%
- Dark / Mid Gray — 15%
- Brand Blue — 10% (reserved for the single most important element)

**Rules:**
- Never mix all three blues in one composition — choose one dominant blue
- Brand Blue is a voice, not a wallpaper — use for headlines, buttons, hero surfaces
- **Never use Brand Blue for body text under 18px**
- Occasionally: reverse the ratio for hero pieces (full Brand Blue cover + white lockup)

## Typography

A serif that carries weight. A sans that carries clarity.

### Typefaces

| Role | Font | Fallback |
|------|------|---------|
| **Display / Headings** | Clarendon URW (Extended Wide) | Roboto Slab → Georgia |
| **Body / UI / Captions** | CMG Sans | Source Sans 3 → Helvetica |

- **Clarendon URW** is the licensed display face. Use Roboto Slab when the license is unavailable.
- **CMG Sans** is the licensed text face. Use Source Sans 3 when CMG Sans is unavailable.

### Type Scale

| Level | Size | Font | Weight | Leading |
|-------|------|------|--------|---------|
| Display | 72pt | Clarendon URW | 500 | 1.0 |
| Heading 1 | 48pt | Clarendon URW | 500 | 1.05 |
| Heading 2 | 30pt | Clarendon URW | 500 | 1.15 |
| Subhead | 20pt | CMG Sans | 600 | 1.25 |
| Body | 16–18pt print / 17–20px screen | CMG Sans | 400 | 1.55 |
| Caption | 10pt | CMG Sans | 600 | tracked |

**Screen body:** 17–20px, line height 1.5–1.6.

### Typography Rules

**Do:**
- Pair Clarendon for headlines, CMG Sans for body — always
- Use italics sparingly — tagline emphasis and pull quotes only, never decoration
- Track all-caps settings: 0.14–0.22em letter-spacing at 12pt and below
- Minimum 12pt for print body; 16px minimum for screen body

**Don't:**
- Set body copy in Clarendon (serif reserved for headings only)
- Mix other serifs (no Times, Cambria, etc.) alongside Clarendon URW
- Stretch or condense letterforms — use the cut as drawn

## Logo & Mark

The primary mark is a woven cross — four interlaced arcs reading as a cross, open hands, and a window.

### Primary Lockup (4 components)
1. **The Mark** — Four interlaced arcs forming a cross. Never modify, rotate, or recolor outside approved variants.
2. **The Wordmark** — "First Presbyterian" in display serif, bold, two fixed lines, ranged left.
3. **The Qualifier** — "Church of Griffin" in all-caps sans, spaced to wordmark width. Never omit at sizes > 180px.
4. **Optical Spacing** — Mark sits left of wordmark with gap = ½ mark width. Not adjustable.

**Lockup ratio: 2:1**

### 3 Approved Variants

| Variant | Background | Use Case |
|---------|-----------|----------|
| **Primary · Light** (default) | White, warm neutrals, Light Gray `#D9E1E4` | Default usage |
| **Reversed · Dark** | Dark `#29292A`, Brand Blue, photos ≥60% dark coverage | Low-light |
| **Reversed · Brand** | Brand Blue `#195C79` (white lockup) | Bulletins, banners, Instagram hero |

### Clear Space (X = height of the cross icon)

- **1X** — Minimum isolation. No type, image, rule, or edge may enter.
- **2X** — Preferred for bulletin covers, posters, banners.
- **½X** — Tight digital headers only, with written approval. Never beside other logos.

### Minimum Sizes

| Usage | Size |
|-------|------|
| Hero / Posters | ≥ 480px / 4.0 in |
| Bulletins / Web | 220–479px |
| App / Favicon | 140–219px (switch to mark-only below 140px) |

### Logo Misuse — Never:
- Stretch or distort the lockup ratio
- Rotate the lockup
- Recolor outside approved variants
- Add drop shadows, bevels, glows, or 3D effects
- Place on low-contrast photos (use reversed variant)
- Place on off-brand color fields
- Outline or stroke the mark (it is solid fill only)
- Place on busy patterns

## Imagery

Natural, warm, unstaged.

**Preferred subjects:** Worship (wide), candid community moments, hands/detail, sanctuary architecture, table/fellowship, portraits in available light.

**Rules:**
- Prefer congregation photos over stock imagery
- Use available light and real moments
- Faces over logos, always

**Avoid:** Awkward posing, corporate stock, heavy filters, teal-and-orange color grades, worship lighting that flattens the scene.

## Bundled Font Assets

All brand fonts are included in `assets/fonts/`. **Always install and use these before rendering any artifact** — do not fall back to Georgia/Helvetica if the fonts are available.

### Font Directories
- `assets/fonts/clarendon-urw/` — 50 OTF files across 5 cuts (Standard, Narrow, Wide, Extended Narrow, Extended Wide)
- `assets/fonts/cmg-sans/` — 32 TTF files (standard + condensed, all weights)

### Install Before Use (Linux/bash)
```bash
cp assets/fonts/clarendon-urw/*.otf /usr/local/share/fonts/
cp assets/fonts/cmg-sans/*.ttf /usr/local/share/fonts/
fc-cache -fv
```

## Quick Reference for Artifact Creation

When building any artifact for FPC Griffin:

1. **Background:** Start with Light `#F4F7F8` or Soft Cream `#FAF6F1` for most pieces
2. **Headlines:** Clarendon URW (or Roboto Slab/Georgia fallback), Brand Blue `#195C79` or Dark `#29292A`
3. **Body:** CMG Sans (or Source Sans 3/Helvetica fallback), Dark `#29292A` or Mid Gray `#54595B`
4. **Accent:** Brand Blue `#195C79` — used sparingly, for the single most important element
5. **Tone:** Warm, relational, hope-forward — point to Christ, not programs
6. **Tagline** when needed: "Know joy. Know love. Know Jesus." — never rearranged
